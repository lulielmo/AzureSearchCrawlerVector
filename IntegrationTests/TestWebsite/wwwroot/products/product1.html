Webbcrawler Pro Edition

9999 kr

Webbcrawler Pro Edition är en avancerad lösning för företag som behöver indexera stora mängder webbinnehåll. Med stöd för parallell bearbetning och intelligent resurshantering kan denna crawler hantera även de mest krävande uppgifterna.

Funktioner

  • Stöd för parallell crawling
  • Inbyggd respekt för robots.txt
  • Intelligent köhantering
  • Automatisk återhämtning vid fel
  • Detaljerad loggning och övervakning
  • Azure Search integration

Tekniska specifikationer

  • Utvecklad i .NET 8
  • Stöd för olika innehållstyper
  • Konfigurerbar minnesanvändning
  • REST API för integration
  • Omfattande dokumentation

Systemkrav

  • Windows 10/11 eller Linux
  • Minimum 8GB RAM
  • .NET 8 Runtime
  • Internet-anslutning
